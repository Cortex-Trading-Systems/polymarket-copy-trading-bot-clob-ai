# Contributing to Cortex Polymarket Copy-Trading Bot

First off, thank you for considering contributing to Cortex AI! It's people like you that make the open-source crypto community such a great place.

## How Can I Contribute?

### Reporting Bugs
* Use the **GitHub Issues** tab to report bugs.
* Describe the steps to reproduce the issue and include your environment details (Python version, OS).

### Suggesting Enhancements
* Open an issue with the tag `enhancement`.
* Explain why this feature would be useful for Polymarket arbitrage or copy-trading.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the documentation (README) is updated if you change functionality.
4. Issue a pull request!

## Style Guide
* We follow PEP 8 for Python code.
* Use descriptive commit messages.
* For AI-related modules, ensure your logic is documented in the comments.

## Community
By participating in this project, you agree to abide by our code of conduct and maintain a professional environment for all traders and developers.
